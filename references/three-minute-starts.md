# Three-Minute Starts

Use this reference when the user asks to start a task, recover after falling off, or reduce resistance.

## Template

```text
今天先不追求完整，只追求启动。

任务：
阻塞类型：
下一步物理动作：
3 分钟启动：
5 分钟扩展：
今天做到什么算赢：
如果卡住：
```

## Common Conversions

| Vague Todo | 3-Minute Start |
|---|---|
| 写文章 | 打开文档，写 5 个标题候选 |
| 做报告 | 新建文件，列出听众和 3 个问题 |
| 学英语 | 打开材料，读 1 段并标 1 个不会的词 |
| 整理房间 | 拿一个袋子，只收桌面上的垃圾 |
| 健身 | 换鞋，走到门口或做 5 个深蹲 |
| 联系别人 | 写出一句开场白，不要求发送 |
| 研究方案 | 打开一个资料，摘 3 个要点 |

## Blocker Labels

- `unclear`: 任务不知道具体做什么。
- `too_big`: 任务太大，启动成本高。
- `emotional_resistance`: 羞耻、害怕、厌烦、压力阻断行动。
- `dependency`: 等别人、等资料、等决策。
- `energy_mismatch`: 当前精力不适合任务大小。
- `perfectionism`: 想一次做好，所以迟迟不开始。
- `waiting`: 已经做完自己的部分，需要标记等待。

## Recovery Script

```text
断掉不等于失败，只说明系统需要更小。
现在只回到一个动作：打开/拿出/写下/发送/列出。
做 3 分钟后停下，记录一句：我推进了什么？下一步是什么？
```
